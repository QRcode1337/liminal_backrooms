Iosevka Term Font Files
========================

This directory should contain Iosevka Term font files for optimal ASCII art rendering.

Required Files:
--------------
- IosevkaTerm-Regular.ttf
- IosevkaTerm-Bold.ttf (optional)
- IosevkaTerm-Italic.ttf (optional)

Where to Get Iosevka Term:
--------------------------
1. Visit: https://github.com/be5invis/Iosevka/releases
2. Download the latest "PkgTTF-IosevkaTerm-*.zip" file
3. Extract the .ttf files from the "TTF" folder
4. Copy them into this directory

Alternative: Using System Fonts
-------------------------------
If you don't add the font files here, the application will fall back to:
- Consolas (Windows)
- Monaco (macOS)
- System monospace font (Linux)

The application will still work fine, but Iosevka Term provides the best
experience for ASCII art and unicode characters.
